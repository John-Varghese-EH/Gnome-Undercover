## How to Contribute

### Reporting Bugs
1. Check if the issue already exists in [Issues](https://github.com/John-Varghese-EH/Gnome-Undercover/issues)
2. Create a new issue with:
   - Your Linux distribution and version
   - GNOME Shell version (`gnome-shell --version`)
   - Steps to reproduce the bug
   - Expected vs actual behavior

### Feature Requests
Open an issue describing:
- The feature you'd like
- Why it would be useful
- Any implementation ideas

### Pull Requests
1. Fork the repository
2. Create a feature branch: `git checkout -b feature/my-feature`
3. Make your changes
4. Test on your local machine
5. Commit with clear messages: `git commit -m "feat: add new feature"`
6. Push and open a PR

### Code Style
- **Bash**: Follow [Google Shell Style Guide](https://google.github.io/styleguide/shellguide.html)
- **Python**: Follow PEP 8
- Use meaningful variable names
- Add comments for complex logic

### Testing
- Test on multiple package managers (apt, pacman, dnf, zypper)
- Test theme switching works correctly
- Verify uninstall cleans everything up
